# Пояснение к датасету:
1. ticker - наименование позиции на бирже
2. commodity - обозначение позиции человеческим языком
3. date - дата записи YYYY-MM-DD
4. open - цена позиции в начале дня
5. low - самая низкая цена за день
6. high - самая высокая цена за день
7. close - цена позиции в конце дня
8. volume - кол-во заключенных сделок за день
